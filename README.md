
# Quick-Shopsy

**Quick Shopsy** is a React e-commerce frontend clone featuring product pages for **Kids, Men, Electronics, and Trending items**. It includes **per-page add-to-cart functionality**, **dark mode**, **responsive design**, and **smooth navigation** for an enhanced shopping experience.

---

## Features

- Multiple product pages: **Kids Wear, Men’s Wear, Electronics, Trending Products**
- **Add to Cart** functionality per product page
- **Dark Mode** support
- **Responsive Design** for mobile, tablet, and desktop
- **Smooth navigation** with React Router
- Quantity-based price calculation
- Individual product details and cart overview
- Styled with **Tailwind CSS**
- Icons powered by **React Icons**
- Scroll animations for product listings using **AOS (Animate On Scroll)**

---

## Techniques Used

- **React** – Frontend library for building UI components  
- **React Router** – For multi-page navigation  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **React Icons** – For all icons including cart, stars, and dark mode toggle  
- **AOS (Animate On Scroll)** – Smooth scroll animations for product sections
